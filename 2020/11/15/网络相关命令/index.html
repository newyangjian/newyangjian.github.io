网络-vim-相关命令

有关网络相关的命令

wget - 网络下载器
wget https: //www.sohu.com/
wget https: //www.sohu.com/ -O sohu.html
scp - 安全的文件拷贝（跨网络拷贝文件）
例如：scp sohu.html root@102.35.73:root/code/
sftp - 安全的文件传输
ping - 网络可达性检测
Ping to Death
分布式拒绝服务攻击：Distributed Deny of Service
DDoS高防服务（防D）—-> 20w
ifconfig / ip —> 本机网络信息
netstat - 查看网络端口使用状况
-n:以数值的方式查询网络端口信息
-l:（监听状态）
-t:TCP(只查看TCP协议的链接)
-p:process(进程)
ssh —> 安全远程连接
如果公司有多台云服务器，可以从一台服务器连接到其他服务器
一般情况下，多台服务器中只有一台允许从公网直接访问，其他
服务器都要通过这台服务器使用ssh的方式进行连接，不允许从
公网直接访问，相当于做了物理隔离，而这台可以从公网通过
SSH直接访问的服务器，我们通常称之为跳板机（跳板服务器）。
ssh root@47.104.31.138

vim相关命令

命令模式：所有的按键代表一个操作命令
进入编辑模式: i / I / a / A / o / O
移动光标:
h / j / k / l —> 配合数字使用
0 / $ / w —> 行首 / 行尾 / 下一个单词
G / gg —> 把光标移动到指定的行
Ctrl+e / Ctrl+y —> 翻行
Ctrl+f / Ctrl+b —> 翻页
编辑内容:
dd / d0 / d$ / dw —> 删除
u / Ctrl+r —> 撤销 / 重做
yy / p —> 复制 / 黏贴
保存退出：ZZ
编辑模式：可以编辑内容
Esc 回到命令模式
末行模式：在最下面的一行进行操作（底线命令模式）
：/ ?
退出：q / q! / wq / qa
查找：/关键词 ?关键词
n：正向搜索
N：反向搜索
替换：1,$ s/新闻/狗屎/
c - confirm
g - global
i - ignore case
e - erro
设置：
set nu / set nonu - 行号
set hsl/ set nohls - 高亮
set ruler / set noruler - 标尺
set ts=4 - 制表键的空格数
set expandtab - 把制表键变成空格
set autoindent - 自动缩进
syntax off / syntax on - 语法高亮
映射快捷键：
map <快捷键> ggGddi 命令模式下的快捷键
imap <快捷键> if name==”main“: 编辑模式下的快捷键
inoremap _main if name == ‘main‘:
查看打开的文件
ls
b 编号
拆分屏幕窗口
vs / sp
ctrl+w 切换窗口

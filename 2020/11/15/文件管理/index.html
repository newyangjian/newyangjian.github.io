文件管理

pwd (print working directory) - 显示当前工作目录
以root身份登录：/root
以wagndachui身份登录：/home/wagndachui
cd (change directory) - 切换工作目录
绝对路径: /
相对路径以当前路径为基础
/root - 超级管理员的用户主目录
/home - 普通用户的用户主目录
/etc - 配置文件
/usr - 可执行的命令以及用户安装的程序
/var - 程序运行中产生的数据和日志
/proc - 进程文件
ls - 显示目录先的所有文件
-a 显示所有包括带.的文件
-l 长格式查看
-：文件
d:目录
l:软连接
cat - 查看文件内容 查看文件：cat / tac / head / tail / more / less / od / rev
-n: 显示行号
mkdir - 创建文件夹
rmdir - 删除空文件夹
touch - 创建空文件或者修改文件的最后访问时间
rm - 删除文件或者文件夹
-i: 交互式删除
-r: 递归式删除
-f: 强制删除
cp - 赋值拷贝文件
mv - 移动文件、在相同文件下移动文件相当于重命名
如果需要批量修改文件名（或后缀）可以使用rename
file - 查看文件的性质
wc - 统计文件的字符数 单词数 行数
-l:行数
-w: 单词数
-c: 字符数
uniq - 去重
-c: 统计重复的次数
sort - 排序
-r: 反转
-n: 按数值大小排序
iconv - 编码转换
-f ：原始编码
-t ：目标编码
gb2312 - 国标码（常用汉字、平假名、片假名）
gbk - 国标码扩展码
gb18030 - 收录了汉字、少数民族文字
Unicode - 万国码 - 字符集
utf-8
a - 一个字节
骆 - 三个字节
🦁 - 四个字节（Emoji字符）
utf-16
utf-32

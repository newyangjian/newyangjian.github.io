文件管理

pwd (print working directory) - 显示当前工作目录
以root身份登录：/root
以wagndachui身份登录：/home/wagndachui
cd (change directory) - 切换工作目录
绝对路径: /
相对路径以当前路径为基础
/root - 超级管理员的用户主目录
/home - 普通用户的用户主目录
/etc - 配置文件
/usr - 可执行的命令以及用户安装的程序
/var - 程序运行中产生的数据和日志
/proc - 进程文件
ls - 显示目录先的所有文件
-a 显示所有包括带.的文件
-l 长格式查看
-：文件
d:目录
l:软连接
cat - 查看文件内容 查看文件：cat / tac / head / tail / more / less / od / rev
-n: 显示行号
mkdir - 创建文件夹
rmdir - 删除空文件夹
touch - 创建空文件或者修改文件的最后访问时间
rm - 删除文件或者文件夹
-i: 交互式删除
-r: 递归式删除
-f: 强制删除
cp - 赋值拷贝文件
mv - 移动文件、在相同文件下移动文件相当于重命名
如果需要批量修改文件名（或后缀）可以使用rename
file - 查看文件的性质
wc - 统计文件的字符数 单词数 行数
-l:行数
-w: 单词数
-c: 字符数
uniq - 去重
-c: 统计重复的次数
sort - 排序
-r: 反转
-n: 按数值大小排序
iconv - 编码转换
-f ：原始编码
-t ：目标编码
gb2312 - 国标码（常用汉字、平假名、片假名）
gbk - 国标码扩展码
gb18030 - 收录了汉字、少数民族文字
Unicode - 万国码 - 字符集
utf-8
a - 一个字节
骆 - 三个字节
🦁 - 四个字节（Emoji字符）
utf-16
utf-32

玩转Linux系统-git使用-hexo博客搭建

Linux命令：命令[参数1 参数2 …] [命令作用的对象]

who / w - 查看谁登录了系统
-q：统计登录人数
clear - 清除屏幕上的输出
cal - 查看日历
-3：显示前后3个月的日历
date - 查看时间日期
whatis - 查看命令的描述
whereis - 获取命令的路径
shutdown - 关机或重启
ogout / exit - 退出登录
ps - 查看进程信息
history - 查看历史命令
！历史命名的编号：重新执行历史命名
-c 清楚历史命令
write - 给其他登陆系统的用户发消息
wall - 给所有人发消息

Linux系统获取帮助
man - 查看手册
–help - 查看命令的帮助
直接搜索在线文档：https://man.linuxde.net/
tldr - 三方查看命令帮助的工具（特别人性化）

Git安装（源代码构建安装）

下载：https://mirrors.edge.kernel.org/pub/software/scm/git/git-2.26.2.tar.gz
解压缩和解归档：tar -zxvf <文件名>
进入文件夹： cd git-2.26.2
安装前的配置：./configure –prefix=/usr/local/
补充依赖项：yum install -y libcurl libcurl-devel
构建安装：make && make install
检查：git –version
——————————–git的使用————————————-
初始化（创建本地版本控制仓库） git init
添加文件到仓库的暂存区：git add
git add .
从暂存区移除文件：git rm –cached
查看版本控制状态：git status
配置用户名和邮箱：git config --global user.email "yangjian@163.com"
git config --global user.name "yangjian"
将暂存区的文件同步到本地仓库：git commit -m “项目初始版本”
查看日志：git log
版本回退：git reset –hard d65a6d
git reflog
——————————–绑定远端服务器（公司的私服）—————————
查看：git remote -v
添加：git remote add origin <git私服地址>
配公钥：ssh-keygen -t rsa -b 2048 -C "yangjian@613.com"
将代码上推倒服务器：git push -u origin master
从服务器上克隆到本地：git clone <git私服地址>
获取服务器上的代码：git pull

————————— hexo搭建博客-先安装node.js环境————————–
下载：wget https://nodejs.org/dist/v14.15.0/node-v14.15.0-linux-x64.tar.xz
解压缩：xz -d node-v14.15.0-linux-x64.tar.xz
解归档：tar -xvf node-v14.15.0-linux-x64.tar
配置环境变量：cd node-v14.15.0-linux-x64/bin
pwd(拿到路径)
回到主目录：vim .bash_profix (对它进行修改)
回到主目录先退出一下
检查：node –version

更新npm下载源：npm configure set registry https://registry.npm.taobao.org/
安装hexo：npm install -g hexo-cli
创建博客项目文件：hexo init 文件名
进入目录安装依赖项：cd 文件名
npm install
生成博客：hexo g
运行服务器： hexo s -p 80

###### 托管到GitHub

​ 修改：vim _config.yml
​ 装个插件：npm install hexo-deployer-git --save
​ hexo g -d
